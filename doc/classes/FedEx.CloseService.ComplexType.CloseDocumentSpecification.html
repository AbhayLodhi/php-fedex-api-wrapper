    Contains all data required for close-time documents to be produced in conjunction with a specific set of shipments.

    For January 2010, there are no applicable options for the COD report, the Manifest, or the Multiweight Report (they will only be available in TEXT format). Detail specifications will be added for those report types when customer-selectable options are implemented.
